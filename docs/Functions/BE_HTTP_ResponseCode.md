## BE_HTTP_ResponseCode

    BE_HTTP_ResponseCode

**Description**  

Returns the value of the Response code from the previous call made via any of the BE curl functions. Eg 403 for not found.

**Parameters**

None

**Keywords**  

HTTP Response Code Curl
