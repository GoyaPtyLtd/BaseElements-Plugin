## BE_JSONPath

    BE_JSONPath ( json ; path )

**Description**  

Query json structure using an XPath like syntax. 

**Parameters**

* *json* : the json to parse.
* *path* : the path to the data you want.

**Keywords**  

Removed

* 4.0.0 : Deprecated. Use internal JSON functions instead.
