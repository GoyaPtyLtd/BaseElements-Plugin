## BE_XMLValidate

    BE_XMLValidate ( xmlText ; schemaText )

**Description**  

Validates the XML in *xmlText* against an XSD *schemaText* for correctness.

**Parameters**

* *xmlText* : the XML text to be validated.
* *schemaText* : the XSD schema text to validate against.

**Keywords**  

XML Validate Schema
