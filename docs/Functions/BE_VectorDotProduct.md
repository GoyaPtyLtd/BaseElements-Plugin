## BE_VectorDotProduct

    BE_VectorDotProduct ( a ; b )

**Description**  

Does a dot product of two vectors ( lists ) of numbers.

**Parameters**

* *a* : the first list.
* *b* : the second list.

**Keywords**  

Vector DotProductkeyword

**Version History**

* 4.0.2 : Renamed from BE_Vector_DotProduct
