## BE_TimeUTCMilliseconds

    BE_TimeUTCMilliseconds

**Description**  

Returns UTC time as number of milliseconds - the same as the time from BE_TimeCurrentMilliseconds but in the UTC timezone.

**Parameters**

None

**Keywords**  

Time UTC Milliseconds
